# Bluesky-Discord-Reposter
A simple script that will check an account and post any new images to a discord webhook

## required packages
discord.py
requests
python-dotenv

## How to
edit the .env file with your bluesky user and pass

in settings.json:
  - add the handle you want into the "author" section e.g
      bsky.app - for official bluesky account
      whitericex.bsky.social - for my account
